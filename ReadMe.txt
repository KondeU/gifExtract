User Manual: Copy all the gif files that need to be extracted into this directory, and then double-click to run gifExtract.bat to extract automatically, and the extracted frame images will be saved to the folder with the same name as the gif.

应用描述：gif帧提取器
开源协议：BSD
使用方法：将需要提取帧的所有gif文件拷贝到该目录下，双击运行gifExtract.bat即自动提取，提取到的帧图片将保存到与gif同名的文件夹下。
